Displey turlari

  • Block
  • Inline
  • Inline-Block
  • None

Block - Ajdodiga nisbatan 100% kenglik va yangi qator talab qiladigan va css hossalarini ozida jamlay oladigan elementga aytiladi

Inline - oziga yetarlicha joy qabul qiladi, yangi qator talab qilmaydigan margin, padding xossalarini o'zida jam qilmaydigan elementga aytiladi

Inline-Block - O'ziga yetarlicha joy qabul qiladi, yangi qator talab qilmaydigan va margin, padding xossalarini o'zida jamlay oladigan elementga aytiladi

None - Elementni ko'rinmasa qiladi, element sahifada hisobga olinmaydi
